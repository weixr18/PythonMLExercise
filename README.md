# PythonMLExercise
一些python写的Machine Learning代码。可供学习者参考。

## 题目

- 每一个代码文件对应一道题目。
- 每个代码文件均可独立运行。
- 目前共有16道题目，持续更新中。
- 题目描述参见[这里](https://github.com/weixr18/PythonMLExercise/blob/master/questions.md)

## 数据

- 需手动将data.rar解压到data/
- 所有用到的数据都在data/中.代码通过相对路径访问这些数据，无需额外下载。
